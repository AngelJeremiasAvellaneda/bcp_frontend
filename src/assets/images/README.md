## 📌 Notas de uso

- **Formatos aceptados**: `.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`
- **Peso máximo recomendado**: 200KB por imagen (usa [squoosh.app](https://squoosh.app) para comprimir)
- **Modo oscuro**: Las imágenes se usan igual en ambos modos. Si necesitas variante oscura, agrega sufijo `-dark` (ej: `hero-depa-dark.png`)
- **Placeholder automático**: Si el archivo no existe, el sistema muestra un diseño SVG con colores BCP
